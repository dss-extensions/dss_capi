V 9.0.0
OpenDSS v 9 is a commemorative edition built to celebrate the 100k downloads for OpenDSS. In this version several improvements to the program have been made, including:

- PVSystem2, Storage2, InvControl2 and StorageController2 are part of the base object suite in OpenDSS, deprecating their predecessors. These objects are now PVSystem, Storage, InvControl and StorageControler.
- Includes the seasonal ratings feature for all PDElements.
- Solves minor issues found in interfaces for circuit breakers.
- Adds interfaces to PVSystem interface for getting instantaneous irradiance values.
- Warning messages restored while compiling the model by adding pop-up windows when the error occurs, complementing the information provided at the results palette.
- Added compatibility with OpenDSS-GIS.
- Fixes the bug found when clearing memory and the model didn’t include energy meters.
- Implements the framework to improve the A-Diakoptics suite (to be released soon).
- implements weighing factors for improving the automatic tearing algorithms.
